Learning Specialist Toolkit

The new hire onboarding process for Learning Specialists previously consisted of an outdated handover binder followed by shadowing sessions, which is an effective means but not a self-serve means to learn. Using a multimodal approach, I developped a website to host the Learning Specialist onboarding material that can be easily updated. To accelerate onboarding, I also co-created video tutorials for our in-house accommodation software called Ventus.

Created using HTML, CSS, Javascript, hosted on Glitch

Academic Accommodations 101 Workshop

The initial Academic Accommodations student onboarding process involved a lot of repetitive information being shared during the intake meeting. I created an online workshop with the goal of engaging students in learning about our processes, procedures, and set expectations of accommodations at the postsecondary level. With the Academic Accommodations 101 workshop now being a pre-requisite as part of our student onboarding, we have been able to reduce intake appointment times by 15 minutes. This has also allowed the team of Learning Specialists to see more students per week during peak busy periods.

Created using HTML, CSS, Javascript, hosted on Glitch. This workshop was initially developped on Brightspace LMS. A version on Articulate Rise 360 is also available.

Adapted Exams Online Workshop

This interactive online workshop was created to teach students registered with Academic Accommodations at the University of Ottawa the ins and outs of the Exam Centre. In addition, I developped this asynchronous workshop with a goal of reducing the workload for Learning Specialists and Exam Coordinators from the Accommodations team and set clear expectations for students using the Exam Centre. This workshop is completed by students prior to their first exam with the Academic Accommodations Service.

Created using Articulate Rise 360
Also available in French

Git & GitHub: A Guide for Beginners

As a developer, you'll need to familiarize yourself with Git version control and GitHub whether you're working professionally in tech or just starting your journey. With so much documentation out there it can be easy to feel overwhelmed. This self-serve online workshop includes must-know content and is meant to teach you the basics of Git and GitHub in an interactive way.

Created using Articulate Rise 360
